# Quick Reference Card -- PreRollTracker

Print this page and keep it near your workstation for easy access.

---

## Website Address

In the address bar at the top of your browser, type: **himomstats.online**

---

## Log In

1. Go to **himomstats.online**.
2. Type the password in the **Password** box.
3. Leave **"Remember me for 30 days"** checked.
4. Click the green **Login** button.

---

## Update Production Counts

1. On the dashboard, click or tap the batch you are working on.
2. Find the count fields: **0.5g**, **0.7g**, **1.0g**.
3. Type the **total** number of pre-rolls made so far for each size.
4. Counts save automatically.
5. Click **Back** or use the navigation to return to the dashboard.

---

## Change a Batch Stage

1. Click or tap the batch on the dashboard.
2. Find the **Stage** dropdown.
3. Select the next stage.
4. Click **Save**.

---

## Check Paper Inventory

1. Click **Paper Inventory** in the navigation bar (or open the menu on a phone).
2. Check the **Total Papers** for each size.
3. If any size shows a low stock warning, tell your supervisor.

---

## Update Paper Inventory

1. Go to **Paper Inventory**.
2. Enter the number of **Boxes** and **Individual Papers** for the size you are updating.
3. Click **Save** or **Update**.

---

## View Finished Goods

1. Click **Finished Goods** in the navigation bar.
2. Find the METRC package by scrolling or using the **Search** bar.
3. Check available quantities and SKU breakdown.

---

## Create a Wholesale Hold

1. Click **Wholesale** in the navigation bar.
2. Find the product and click **Hold** or **Reserve**.
3. Enter the **Quantity** and any **Notes**.
4. Click **Create Hold**.

---

## Batch Stage Reference

| # | Stage | Color | Meaning |
|---|---|---|---|
| 0 | Not ground | Brown | Flower received, not yet ground |
| 1 | Ground | Orange | Flower ground, ready to roll |
| 2 | In progress | Blue | Actively rolling pre-rolls |
| 3 | Finished | Purple | Rolling complete |
| 4 | Waiting on testing | Red | Sent to lab, waiting for results |
| 5 | Test passed | Green | Lab results passed |
| 6 | Bagging | Deep orange | Packaging in progress |
| 7 | Done | Dark green | Complete, ready for sale |

## Common Problems -- Quick Fixes

| Problem | Fix |
|---|---|
| Can't log in | Check Caps Lock, retype password carefully |
| Page won't load | Check Wi-Fi, try refreshing |
| Counts didn't save | Check internet, re-enter counts |
| Page looks broken | Press Ctrl+Shift+R (or Cmd+Shift+R on Mac) |
| Forgot password | Click "Forgot password?" on login page, use recovery key |

---

## Navigation Quick Reference

| Button | What It Does |
|---|---|
| **+ Add Strain** | Create a new batch |
| **Finished Goods** | View completed METRC packages |
| **Wholesale** | Manage wholesale inventory holds |
| **Paper Inventory** | Check and update paper/cone supplies |
| **Analytics** | View production charts and statistics |
| **Archive** | View completed past batches |
| **Settings** | Change password and app settings |
| **Audit** | View history of all changes |
| **Logout** | Sign out of the app |

---

## On a Phone

- Tap the **three-line menu button** (top left) to see all navigation options.
- Tap the dark area or the **X** to close the menu.
- Tap a batch card to open it for editing.
